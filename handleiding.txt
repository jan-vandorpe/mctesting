*******************
*** INSTALLATIE ***
*******************

1. GIT CLONE

	Kloon het project met behulp van GIT in de root folder van de webserver.
	(vb. c:\xampp\htdocs; na klonen het je het project dus in de folder c:\xampp\htdocs\mctesting)


2. DATABASE

	Importeer SQL bestand in MySQL.

	SQL bestand staat op de root van de project folder. (..\mctesting\mctesting.sql)

3. CONFIGURATIE BESTAND

	Pas in het configuratie bestand de waarden voor host, databasenaam, user en paswoord aan waar nodig.

	Configuratie bestand: ..\mctesting\src\McTesting\Config\Config.php
